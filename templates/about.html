About MedVision Medical Assistant

MedVision Medical Assistant is a research-oriented tool designed to assist clinicians, radiologist and researchers by providing AI-guided image analysis. Upload medical reports and receive probable diagnoses along with guidance — always verify with a qualified medical professional.

How it works

  • Client-side resizing & secure upload of images
  • Session-based chat memory to enable follow-up queries
  • Calls a hosted LLM (Groq) to process image + text prompts
  • Conservative responses: models indicate uncertainty and recommend clinical confirmation

Open Source / Privacy

This project stores session data in-memory by default. For production, we recommend using encrypted persistent storage and secure environment variables for API keys. Please do not upload patient-identifiable data unless you have appropriate consent and safeguards in place.
